# Programação para Automação de Testes

Repositório com a implementação do trabalho final da disciplina **Programação para Automação de Testes** da Pós-Graduação em automação de testes de software, utilizando JavaScript com Mocha e Node.js.

## Como rodar os testes

1. Instale as dependências (caso ainda não tenha):

```bash
npm install
```

2. Execute os testes com:
```bash
npm test
